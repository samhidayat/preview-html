BBM-AN DI PC / LAPTOP KENAPA GAK ?
1. Download, extract dan install bluestack android terbaru di
http://www.alyaku.tk/2014/07/bbm-di-pc-laptop-kenapa-gak.html
Cara instalnya sama saja seperti install software lain, SETELAH SUKSES INSTAL,
nanti ada jendela bluestack sedang loading, close saja dan restart PC
2. Setelah PC di restart, klik kanan icon bluestack di pojok kanan taskbar anda, dan pilih
menu restart agar aplikasi bluestack restart dengan sendirinya.
